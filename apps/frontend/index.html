Selamat Datang di Salahuddin Library

Perpustakaan digital di Banda Aceh yang menyediakan katalog buku online, layanan peminjaman, program edukasi untuk anak-anak, dan donasi buku. Tempat belajar, berbagi ilmu, dan menumbuhkan cinta membaca untuk generasi masa depan.

Program Kami

  • Children Read Out-Loud - Aktivitas membaca nyaring untuk anak
  • IT Class - Belajar Microsoft Office dan keterampilan digital
  • Story Telling Class - Belajar teknik bercerita yang menarik
  • Public Speaking Class - Menjadi public speaker profesional

Katalog Buku

Temukan berbagai koleksi buku berkualitas untuk pengembangan diri dan pengetahuan di Salahuddin Library.

Donasi Buku

Berbagi ilmu melalui buku untuk generasi penerus bangsa. Buku Anda akan dibaca oleh puluhan anak-anak.
